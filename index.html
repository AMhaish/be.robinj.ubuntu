Ubuntu Launcher is still a work in progress, and can be found on Google Play.

If Beta isn't bleeding-edge enough for you, you are free to join the Google+ community, after which you can use this link to opt-in for alpha-testing. You should then be able to get the latest alpha version through Google Play.

Feedback and bug reports are always welcome on GitHub (preferably), the Google+ community or via e-mail.
